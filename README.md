### Usage
##### Folder Tree
>dataset
>> mask
>> img
>input
>output
>> preprocess
>> slic
>> slic_merge
>> paddy
>> paddy_mask
>> seedlings
>>> img
>>> position

##### build the dataset
    The labeled binary imgs in `./dataset/mask/` should be naming like: **`label`\_`imgName`.jpg**
    , where white pixels means the labeled area.
